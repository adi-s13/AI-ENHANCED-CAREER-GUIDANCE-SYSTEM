# AI Career Guidance System
It is a career recommendation system that uses:
- A psychometric test (25 questions)
- The student's 10th and 12th marks
- Sentence-BERT (SBERT) text embeddings
- An enriched dataset of career information

The backend is built using **Flask**, and **MongoDB Atlas** is used to store user data.

---

## Features
- User signup/login
- Psychometric test with 7 trait scores
- Academic marks input
- Hybrid recommendation model (SBERT + traits + marks)
- Final career recommendations with explanation
- Auto-generated career report

---

## Tech Used
- Python  
- Flask  
- MongoDB  
- SBERT (all-MiniLM-L6-v2)  
- HTML / CSS / JS
